# Projet_2025_Chess_DB

Le but du projet est de réaliser une application desktop de gestion de matchs d'une fédération d'échecs.
Dans cette application, on retrouve trois fenêtres : Dashboard, Player et Tournament. Le dashboard est la page d'accueil du gestionnaire. La fenêtre Player permet la gestion des joueurs : inscrition, modifications des informations ainsi que la suppresion.
Tournament contient une partie de gestion de tournois (création, modification et suppression du tournois) et une partie d'ajout de compétitions.
Pour pouvoir ajouter un joueur dans un tournois, on clique sur le bouton jaune et une petite fenetre apparait. Sur celle-ci, on insére le matricule du joueur qui participe au tournois et on voit la liste des matricules des joueurs déjà inscrit. On dispose aussi d'une barre de recherche pour faciliter la gestion des joueurs inscrit.
Pour pouvoir ajouter une compétition, il faut d'abord sélectionner le tournois ou a été joué la compétition. En sélectionnant un tournois, la liste des compétitions apparait.
